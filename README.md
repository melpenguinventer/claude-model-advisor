# Claude Model Advisor

An interactive tool to help you choose the right Claude model for your task and get an optimized prompt.

## Features
- **Smart recommendations** based on task complexity, speed, budget, and accuracy needs
- **Cost-benefit analysis** to ensure you're not overpaying for capability
- **Auto-generated prompts** optimized for your chosen model
- **Complexity helper** for when you're unsure about task difficulty

## Models Covered
- **Claude Haiku 4.5** - Fast and affordable
- **Claude Sonnet 4.6** - Balanced workhorse  
- **Claude Opus 4.6** - Deep reasoning for complex problems

## How to Use
1. Open `index.html` in your browser
2. Answer questions about your task
3. Get a model recommendation with cost breakdown
4. Copy the generated prompt to use in Claude

## Try It Online
[Visit the live tool](https://melpenguinventer.github.io/claude-model-advisor)
